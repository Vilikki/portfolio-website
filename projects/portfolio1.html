Race Interface

A real-time race management and information system for racing events. The application allows employees to manage race sessions, track laps, control race flags, and display live race information to drivers and spectators.

Built with JavaScript, Node.js, Express, and Socket.IO for real-time communication between clients and the server. Employee interfaces are protected using access keys stored in environment variables.

Features

  • Real-time race management using Socket.IO
  • Multiple synchronized public display views
  • Separate employee interfaces with access control
  • Live leaderboard
  • Race flag control system
  • Responsive layouts for desktop and tablet use
  • Fullscreen support for public race displays
  • Automatically saves race state to disk, allowing the system to restore and resume
