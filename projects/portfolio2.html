Cars Viewer

A website that showcases information about different car models, their specifications, manufacturers, and more. Built with Go (Golang) using Server-Side Rendering.

Built with Go, html and css. Utilizing go templates, connecting to custom API for car information.

Features

  • Browser inventory
  • Advanced filtering
  • Car recommendations
  • Comparison tool
