Pathfinder

Pathfinding algorithm for finding most efficient paths to move trains from one destination to another with least moves possible.

The program uses multiple pathfinding algorithms, pits them against each other and gives the winner as a result for given train path. Printing out each train moves turn by turn. Features BFS, DFS, AStar and Ants algorithms.

Features

  • Pathfinder identifies the shortest routes from start to end station
  • Pathfinder identifies the quickest way to move some number of trains from start station to end station.
  • Pathfinder identifies and displays minimal number of train movements turns in standard output
  • Generates a visualization of train movements
  • The program exits gracefully irrespective of the inputs
  • The program does not cause memory leaks
