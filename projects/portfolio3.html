Race Interface

Literary Lions Forum is a full-stack web application that allows users to create, discuss, and interact with book-related content.

The application features user authentication, post creation, commenting, category filtering, and a like/dislike system. It is built with Go using a layered architecture, SQLite for data persistence, and server-side rendered HTML templates.

Features

  • HTTP layer (handlers)
  • Server/Data layer (store)
  • Server-side rendered frontend (Go Templates)
  • Database layer (SQLite)
  • Session management (UUID)
  • Docker implementation
  • Like / Dislike functionality
  • Add, remove, edit posts
  • Customizable profile
